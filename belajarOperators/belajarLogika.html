Belajar Logika Operator

Opertor Deskripsi
&& Operator 'Dan' (and) logikanya menghasilkan nilai true apabila kondisinya terpenuhi
|| Operator atau (or) Logikanya akan menghasilkan nilai true apabila ada salah satu kondisi terpenuhi
! Operator tidak (not) Digunakan untuk membalikan suatu kondisi

Contoh Penggunaan Logika Operator
